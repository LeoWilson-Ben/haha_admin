# 玄遇管理后台（Vue 3）

运营管理后台，包含仪表盘、用户管理、名师认证审核、内容审核、提现审核、举报与风控、系统配置。

## 运行

```bash
cd admin
npm install
npm run dev
```

访问 http://localhost:5174 。接口代理到后端 `http://localhost:8000`，需先启动 Django 服务。

## 鉴权

管理端接口需请求头 `X-Admin-Token` 与后端配置的 `ADMIN_API_KEY` 一致。

1. 首次使用：在「系统配置」中填写与服务器一致的管理端密钥，点击保存（存于本地 localStorage）。
2. 后端：生产环境请设置环境变量 `ADMIN_API_KEY`；未设置时仅 DEBUG 模式下允许访问。

## 功能说明

- **仪表盘**：总用户数、帖子数、待审核名师/提现/举报数量，可点击跳转对应列表。
- **用户管理**：用户列表、按手机号/昵称搜索、禁用/启用账号。
- **名师认证审核**：名师入驻申请列表，通过/驳回；通过后该用户出现在 APP「真人名师」列表。
- **内容审核**：帖子列表，按状态筛选、关键词搜索，下架/恢复帖子。
- **订单与财务**：提现申请列表，通过/驳回。
- **举报与风控**：举报列表，填写处理结果并标记已处理。
- **系统配置**：管理端密钥（X-Admin-Token）的保存与说明。
